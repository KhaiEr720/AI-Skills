# Design system defaults (good starting point)

Use these defaults when the project has no explicit design system.

## Spacing

- Use an 8px rhythm (4/8/12/16/24/32/48)
- Prefer generous whitespace over dense clustering

## Typography

- Body text: 14-16px (mobile: 16px is safer)
- Headings: clearly stepped sizes
- Line height: ~1.4-1.6 for body

## Buttons

- Primary: one per view when possible
- Labels: verb + object ("Create project")
- States: default, hover, focus, disabled, loading

## Inputs

- Persistent label + helper text
- Error text below field

## Tables

- Sticky header if long
- Column alignment consistent
- Empty and loading states
